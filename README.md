# OpenSauce  API

## Docs:
The documentation for the API can be found [here](https://opensauce.gitbook.io/opensauce/) the ping endpoints are not documented.

# Commands
Start app with nodemon
```
npm run dev
```
Start App
```
node . 
```
OR 
```
npm start
```
Lint: 
```
eslint . --fix
```
